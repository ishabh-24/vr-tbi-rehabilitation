# Traumatic Brain Injury Rehabilitation Game (Keyboard Version)

---

## **Overview**

The **Traumatic Brain Injury Rehabilitation Game** is designed to support **Traumatic Brain Injury (TBI) patients** through engaging, task-oriented rehabilitation exercises. The game is based on **Constraint-Induced Movement Therapy (CIMT)** principles, focusing on promoting motor recovery while stimulating cognitive functions.

Key goals of the game:
- Reduce anxiety associated with rehabilitation exercises  
- Improve motivation through celebratory feedback after tasks  
- Support both **upper-limb motor function** and **cognitive processing skills**  

The game includes multiple levels, each targeting specific rehabilitation goals, and features **calming background music** to enhance patient comfort and engagement.

---

## **Game Levels**

### **Level 1 – Pouring Task**
- **Task:** Pour pearls from a bottle into a cup  
- **Features:** Reset button for repeated attempts  
- **Focus:** Physical rehabilitation, hand-eye coordination  

### **Level 2 – Object Organization**
- **Task:** Organize colored objects (red, green, blue) onto specific shelves according to instructions  
- **Features:** After three failed attempts, instructions can be reviewed  
- **Focus:** Cognitive and physical rehabilitation, memory, attention, motor control  

### **Level 3 – Laundry Sorting**
- **Task:** Manage two loads of laundry (dark and light clothes) and wait for completion  
- **Features:** Reset button included  
- **Focus:** Cognitive and physical rehabilitation, patience, task sequencing, motor coordination  

---

## **Therapeutic Features**

- **Calming music:** Reduces anxiety during rehabilitation  
- **Celebratory feedback:** Increases motivation after completing each level  
- **Cognitive challenges:** Memory, sequencing, and attention tasks  
- **Physical challenges:** Task-oriented movements for motor recovery

## **Installation and Setup**
1. Clone the repository:  

```bash
git clone https://github.com/ishabh-24/vr-tbi-rehabilitation.git
```
2. Open the project in Unity
```bash
#Navigate to project folder
cd vr-tbi-rehabilitation

#Open Unity (replace with your Unity version path)
open -a "Unity Hub" .
```
3. Play the scenes using keyboard controls
